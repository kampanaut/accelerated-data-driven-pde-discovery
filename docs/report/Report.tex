\IEEEPARstart{P}{artial} differential equations are the primary mathematical
language for modelling spatiotemporal dynamics in physical systems, from fluid
turbulence and chemical pattern formation to heat conduction and
biological signalling. While the governing equations for many systems are well
established, there exist phenomena whose underlying PDEs are unknown or only
partially understood. Data-driven PDE discovery addresses this gap: given
observations of a system's evolution, can we recover the governing equation
directly from pointwise data?

Raissi \cite{raissi2018} introduced the Deep Hidden Physics framework, in which
a neural network, the PDE operator network, learns the nonlinear mapping
from spatial derivatives to the time derivative:
%
\begin{equation}
u_t = \mathcal{N}(u, u_x, u_y, u_{xx}, u_{yy})
\label{eq:pde_discovery}
\end{equation}
%
Given solution data and its derivatives, the network is trained to approximate
the right-hand side of the PDE. Once trained, the learned operator encodes the
discovered physics: its weights implicitly represent the PDE coefficients,
which can be extracted via Jacobian probing.

In Raissi's formulation, two networks are trained in tandem: a \emph{solution
network} that approximates the field $u(\mathbf{x}, t)$ from sparse
observations, and the PDE operator network $\mathcal{N}$ that learns the
right-hand side. The solution network provides spatial derivatives via
automatic differentiation, while the operator network constrains the solution
to satisfy the discovered PDE. When a high-fidelity numerical solver is
available, however, the solution network becomes unnecessary: the solver
provides exact solution fields and their spatial derivatives directly. This
\emph{decoupled} setting --- which Raissi showed to be viable --- isolates the
PDE discovery problem to learning the operator $\mathcal{N}$ alone, and is the
setting adopted in this work.

PARAPHRASE THIS: Representing the nonlinear function N by a deep neural network is the novelty
of the current work. Deep neural networks are a richer family of function
approximators and consequently we do not have to commit to a particular class
of basis functions such as polynomials or sines and cosines. This
expressiveness comes at the cost of losing interpretability of the learned
dynamics. However, there is nothing hindering the use of a particular class of
basis functions in order obtain more interpretable equations [35].

However, this approach requires training a new network from scratch for each
system encountered. Every new set of physical parameters --- a different
viscosity, a different diffusion coefficient --- demands a full training run.
This is expensive and fails to exploit the structural similarities that exist
across related systems. For instance, a network trained on one viscosity of
Navier-Stokes learns representations relevant to nearby viscosities, yet none
of this knowledge transfers to the next training run.

Model-Agnostic Meta-Learning (MAML) \cite{finn2017} offers a solution to this
training problem. Rather than learning a single set of weights for one system,
MAML learns a meta-parameter \emph{initialisation} $\theta^*$ --- a starting point in the loss
landscape from which the network can rapidly adapt to any system within a task
family of one PDE. This solves the expensive training of a large model or scarce
dataset.

This work applies MAML to the decoupled PDE operator network to
\emph{amortise} the discovery of PDE coefficients across a family of related
systems. During meta-training, the algorithm samples tasks from a distribution
of PDE instances --- each characterised by different physical parameters ---
and learns an initialisation $\theta^*$ that minimises the expected
post-adaptation loss across the family. Empirically, the resulting $\theta^*$ encodes a
\emph{prior} over the coefficient space: before any task-specific fine-tuning,
the network already represents a meaningful estimate of the PDE coefficients.
Adaptation then refines this estimate toward the true parameters of the target
system, reducing the per-instance cost from full training to a small number of
gradient steps. The nature of this estimate --- whether it reflects a measure of central
tendency, a minimax strategy over the task distribution, or an artefact of the
loss landscape geometry --- is itself an empirical question examined in this paper.

Formally, let $\mathcal{N}_\theta$ denote the PDE operator network
parameterised by $\theta$. For a PDE family $\rho$, the task distribution
$\tau(\rho)$ generates tasks $\mathcal{T}_i$, each consisting of support and
query sets of $\{(\mathbf{x}_j, \mathbf{y}_j)\}$ pairs, where $\mathbf{x}_j$
are input spatial derivative features and $\mathbf{y}_j$ are target time
derivatives. The MAML inner loop adapts $\theta^*$ to a task-specific
$\theta'_i$ by gradient descent on the support set, traversing the loss
landscape toward the task's local minimum. The outer loop then updates
$\theta^*$ based on the query-set performance of the adapted parameters
$\theta'_i$ across all sampled tasks which gives way to steering the next
initialisation away from regions where task-specific adaptation fails to
generalise.

This work investigates whether MAML can amortise PDE coefficient recovery. 

$$
u_t = D \nabla^2 u \text{ Heat equation}
$$

$$
u_t = D (1-u)\nabla^2 u \text{ Nonlinear Heat equation}
$$

Heat equation and nonlinear heat equation are chosen to be the functions
approximated by the MAML MLP. 

Defining the requirements of amortisation is the objective of this paper. This
work examines how the distribution of the training tasks affects the quality of
amortisation, and what failure modes arise afterwards.

\section{Methodology}

\subsection{PDE Operator Discovery}

Raissi~\cite{raissi2018} considers nonlinear partial differential equations of
the general form
%
\begin{equation}\label{eq:pde_general}
  u_t = \mathcal{N}(t,\, x,\, u,\, u_x,\, u_{xx},\, \ldots),
\end{equation}
%
where $\mathcal{N}$ is an unknown nonlinear function of time~$t$, space~$x$,
the solution~$u$, and its spatial derivatives. Given scattered observations
$\{t^i, x^i, u^i\}_{i=1}^{N}$, the goal is to learn~$\mathcal{N}$ and
thereby discover the governing PDE.

The approach approximates both the unknown solution~$u$ and the nonlinear
function~$\mathcal{N}$ with deep neural networks, and defines a \emph{deep
hidden physics model}
%
\begin{equation}\label{eq:residual}
  f := u_t - \mathcal{N}(t,\, x,\, u,\, u_x,\, u_{xx},\, \ldots).
\end{equation}
%
Derivatives of the neural network~$u$ with respect to time and space are
obtained via automatic differentiation. Parameters of both networks are learned
by minimising the sum of squared errors
%
\begin{equation}\label{eq:raissi_loss}
  \sum_{i=1}^{N}
    \bigl(\lvert u(t^i, x^i) - u^i \rvert^2
    + \lvert f(t^i, x^i) \rvert^2\bigr),
\end{equation}
%
where $\{t^i, x^i, u^i\}_{i=1}^{N}$ denote the training data on~$u$. The
first term fits the data by adjusting the parameters of the neural network~$u$;
the second learns the parameters of the network~$\mathcal{N}$ by trying to
satisfy the PDE at the collocation points $(t^i, x^i)$.

When a numerical solver provides the solution~$u$ and its derivatives
directly, the solution network is unnecessary. In this \emph{decoupled}
setting we train only the operator network, reducing the loss to
%
\begin{equation}\label{eq:decoupled_loss}
  \mathcal{L}(\theta)
  = \frac{1}{N} \sum_{i=1}^{N}
    \bigl\lVert \mathcal{N}_\theta(\mathbf{x}_i) - \mathbf{y}_i \bigr\rVert^2,
\end{equation}
%
where $\mathbf{x}_i$ collects the solver-provided features (field values and
spatial derivatives) and $\mathbf{y}_i$ the time derivatives serving as
targets. This then allows us to amortise the PDE Operator Network
Regularisation Agent with MAML.

\subsection{Meta-Learning via MAML}

Given a distribution of PDE tasks $p(\mathcal{T})$, each characterised by
different physical parameters, MAML seeks an initialisation $\theta$ from
which a small number of gradient steps on any sampled task yields good
generalisation. The procedure is:

\begin{figure}[!h]
\renewcommand{\algorithmicrequire}{\textbf{Require:}}
\begin{algorithmic}[1]
\REQUIRE $p(\mathcal{T})$: distribution over PDE tasks
\REQUIRE $\alpha, \beta$: inner and outer step sizes
\STATE Randomly initialise $\theta$
\WHILE{not converged}
  \STATE Sample batch of tasks $\mathcal{T}_i \sim p(\mathcal{T})$
  \FORALL{$\mathcal{T}_i$}
    \STATE Sample support set $\mathcal{D}_i = \{(\mathbf{x}^{(j)},
           \mathbf{y}^{(j)})\}_{j=1}^{K}$ from $\mathcal{T}_i$
    \STATE Evaluate $\nabla_\theta \mathcal{L}_{\mathcal{T}_i}(f_\theta)$
           using $\mathcal{D}_i$
    \STATE Compute adapted parameters:
           $\theta'_i = \theta - \alpha \nabla_\theta
           \mathcal{L}_{\mathcal{T}_i}(f_\theta)$
    \STATE Sample query set $\mathcal{D}'_i$ from $\mathcal{T}_i$
           (disjoint from $\mathcal{D}_i$)
  \ENDFOR
  \STATE Update $\theta \leftarrow \theta - \beta \nabla_\theta
         \sum_{\mathcal{T}_i} \mathcal{L}_{\mathcal{T}_i}(f_{\theta'_i})$
         using each $\mathcal{D}'_i$
\ENDWHILE
\STATE \textbf{return} $\theta^*$ \COMMENT{meta-learned initialisation}
\end{algorithmic}
\caption{MAML for PDE coefficient recovery (after
Finn et al.~\cite{finn2017}).}
\label{alg:maml}
\end{figure}

The inner loop (lines 5--8) adapts to a single task using $K$~support
samples; the outer loop (line 10) updates~$\theta$ so that the
\emph{post-adaptation} loss on held-out query samples is minimised across all
sampled tasks. The meta-gradient in line~10 differentiates through the inner
update, requiring a second-order gradient computation.
